Models: XGBoost & Random Forest (prediction) and Linear Mixed Model (explanation). Each table shows all 5 data variants for both algorithms. Validation: 2024 = single-split (large data, already reliable); 2025 = 5-fold cross-validation (small data, honest).

How we measured accuracy (single-split vs cross-validation):
  • Single split — train the model on 80% of the grids and test on the other 20% once. Fast, and reliable when there is lots of data (like 2024, 135 grids).
  • 5-fold cross-validation — split the grids into 5 groups, test 5 times (each time holding out a different 20%), then average the 5 scores. This removes the luck of any single split, so it is essential for small data (like 2025, only 24 grids, where one lucky split can look falsely high).
  • The rule: big data → single split is enough; small data → cross-validation gives the honest number. Grids never appear in both train and test, so there is no cheating (no data leakage).
